# README - SUPER-PLUME

Super-plume is a MVP for an online service to personalize easily one's own offer letter in case of property purchase.
To use the service :
1) Go to http://super-plume.herokuapp.com/  
2) File in the form
3) Check your infos 
4) Download your personalized offer letter.
